# Cytech EC

Laravelを用いたECサイト開発課題

## 実装内容
- ユーザー登録機能
- ログイン・ログアウト機能
- リダイレクト制御
- バリデーション（FormRequest）
- バリデーションメッセージ(lang/validation.php)
- 商品一覧表示
- 商品検索機能（名前・価格）
- 商品登録機能（画像アップロード対応）
- 商品詳細画面
- いいね機能（非同期）
- 出品商品詳細機能
- 出品商品編集機能
- 商品更新処理（画像更新対応）
- 売上テーブル追加（購入機能準備）
- マイページ機能
- アカウント編集機能
- 問い合わせ機能

## 環境
- Laravel
- Docker（Sail）
- MySQL
- Mailtrap
